# JeA-NEXT

`JeA` pero hecho con `NEXT13`

## Instalación

Use el administrador de paquetes de [nodejs](https://nodejs.org/en/download/) para desplegar JeA-NEXT.

```node
npm i --force
```

## Uso

```node
npm run dev
```
